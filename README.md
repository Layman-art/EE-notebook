## 📖 项目简介 (Introduction)

欢迎来到我的电气工程（Electrical Engineering）本科核心课程笔记仓库。

这里汇集了我从**大一到大四**的学习轨迹，旨在构建一个系统化的 EE 知识库。内容涵盖了基础数理课程、专业核心课以及高年级的专业选修课。

所有的笔记文件主要分为两类，旨在帮助深入理解和快速复习：
* 📓 **课堂笔记 (Class Notes)**：记录课堂重点、推导过程及平时思考。
* 📝 **期末复习笔记 (Review Notes)**：针对考试整理的知识框架、重点题型和速查表。
